Projeto de TCC para automaçao de controle de temperatura de brassagem de baixo custo para micro cervejarias usando arduino.

O projeto foi concluido e o TCC entregue, contudo o projeto esta aberto à evoluçoes e sugestões de melhoria


Necessário
-Biblioteca pyfirmata
-Programar o arduino com o codio do Firmata
-Comunicação com o arduino o tempo inteiro
-Executar os dois arquivos simultanemente, Arduino Comm.py e Controle de Brassagem 1.0.py
